Overview¶

In this tutorial, we are going to walk you through a whole pipeline of data science, that is, Data Collection and Management, Data Cleaning and Processing, Exploratory Data Analysis, and Further Analysis using machine learning and some statistics knowledge. We are going to show everyone how to create your own COVID-19 Analysis report. There are 4 sections in the tutorial

1. Data Cleaning and Processing

2. Exploratory Data Analysis

3. Linear Regression and Hypothesis Testing

4. Prediction using simple Neural Net

In this process, we are going to use couple of libraries to make our life easier. They are

  • pathlib
  • requests
  • pandas (https://pandas.pydata.org/docs/)
  • numpy (https://numpy.org/doc/)
  • plotnine (https://plotnine.readthedocs.io/en/stable/)
  • matplotlib.pyplot (https://matplotlib.org/3.1.1/api/_as_gen/matplotlib.pyplot.html)
  • statsmodels.formula.api (https://www.statsmodels.org/stable/api.html)
  • sklearn.neural_network.MLPClassifier (https://scikit-learn.org/stable/modules/generated/sklearn.neural_network.MLPClassifier.html)

Intuitively, the answer seems obvious, but as a data scientist, we can perform hypothesis testing to tell whether the result is statistically significant. We will do the following steps.

  1. Have a null hypothesis that there is no relationship between time and number of cases
  2. compute the p-value, the probability that of which oberving an increase under the null assumption.
  3. Reject or accept our null hypothesis

Prediction using simple Neural Nets¶

In the last section, we have shown there is a positive correlation between time and number of COVID-19 cases. However, linear regression failed to give us a precise prediction of future cases. Hence, in this section, we will explore a more powerful tool, Nerual Networks.

As usual, we first need to prepare our dataset. Our goal is to predict the number of cases in future 10 days. To verify how good our model is, we will use all but the last 10 day of data as our training set, the rest as our test set.

    print(f'[{date}]\nTrue: {true:,}\nPrediction: {pred:,}\n Absolute Error: {error}\n Error rate: {error/true:.4f}')
    print('--------')
